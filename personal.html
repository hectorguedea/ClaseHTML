Inicio Acerca
Las principales dificultades asociadas con la gestión de Big Data están vinculadas a su recopilación y almacenamiento, búsqueda, intercambio, análisis y visualización. La Web Semántica proporciona los medios para superar algunos de estos desafíos. Ciertamente, los descubrimientos formales de las tecnologías de la Web Semántica permiten el procesamiento automatizado de datos a través de sofisticadas técnicas de inferencia y razonamiento. RDF (Resource Description Framework) es un modelo estándar para el intercambio de datos en la Web y se basa en gráficos, lo que permite la representación de los datos en forma de triples objeto-predicado. Los tripletes RDF se pueden usar para crear conjuntos de datos y establecer relaciones explícitas entre los datos; esta colección de conjuntos de datos interrelacionados en la Web se conoce como Datos Vinculados. Uno de los objetivos principales de los datos vinculados (abiertos) es agregar una capa semanal sobre los datos, haciéndolo comprensible por las máquinas para que puedan realizar algunas operaciones de análisis de datos en nombre de los usuarios humanos. Así, la Web semántica puede ayudar en el descubrimiento, integración, representación y gestión del conocimiento [30]. En particular, las tecnologías semánticas se han aplicado con éxito en varios escenarios para la integración de datos heterogéneos, el análisis de datos en el nivel de conocimiento y la visualización de datos vinculados.
